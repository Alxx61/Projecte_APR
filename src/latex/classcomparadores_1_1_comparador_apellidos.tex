\doxysubsection{Detailed Description}
Implementación de \mbox{\hyperlink{classcomparadores_1_1_comparator}{Comparator}} para ordenar objetos Publicacion. 

La comparación se basa en el autor de cada publicación que tenga el apellido lexicográficamente menor (el primero en orden alfabético).

Compara dos publicaciones basándose en el primer apellido (según orden alfabético) de sus respectivas listas de autores. 

Para cada publicación (p1 y p2), primero encuentra el autor cuyo apellido sea lexicográficamente el menor. Luego, compara esos dos apellidos resultantes.
