\doxysubsection{Detailed Description}
Interfaz para un componente de búsqueda (Strategy Pattern). 

Define el contrato para cualquier clase que implemente una estrategia de búsqueda sobre un conjunto de publicaciones. Se espera que la clase que implemente esta interfaz almacene internamente los criterios de búsqueda específicos (p.\+ej., un autor, palabras clave, etc.).
