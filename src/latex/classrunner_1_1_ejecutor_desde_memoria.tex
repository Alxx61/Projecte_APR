Ejecuta el escenario de prueba completo de búsqueda y ordenación. 

El proceso detallado es el siguiente\+:
\begin{DoxyEnumerate}
\item Creará un objeto Controlador.
\item Obtendrá las publicaciones invocando al método \mbox{\hyperlink{classgestordatos_1_1_gestor_de_datos_add464c1ea2923974f9a2e382e42bc9ca}{Gestor\+De\+Datos.\+cargar\+\_\+publicaciones\+\_\+de\+\_\+prueba()}}.

\item Realizará una búsqueda de publicaciones por el nombre \char`\"{}\+Geoffrey Hinton\char`\"{}.
\item Generará una lista con las publicaciones encontradas ordenadas por fecha en sentido ascendente.
\item Para cada publicación de la lista generará un String invocando al método {\bfseries{str}}() y lo añadirá al atributo resultado.
\item Generará una lista con las publicaciones encontradas ordenadas por fecha en sentido descendente.
\item Para cada publicación de la lista generará un String y lo añadirá al resultado.
\item Generará una lista con las publicaciones encontradas ordenadas por autor en sentido ascendente.
\item Para cada publicación de la lista generará un String y lo añadirá al resultado.
\item Generará una lista con las publicaciones encontradas ordenadas por autor en sentido descendente.
\item Para cada publicación de la lista generará un String y lo añadirá al resultado.
\item Realizará una búsqueda de publicaciones por las siguientes palabras clave\+: \char`\"{}neural networks\char`\"{} y \char`\"{}optimization\char`\"{} (una sola búsqueda\+: se buscan publicaciones que contengan ambos valores).
\item Repetirá los pasos 5 a 12 para la lista de publicaciones obtenidas.
\item Realizará una búsqueda de publicaciones cuya fecha de publicación esté dentro del intervalo \char`\"{}197001\char`\"{} y \char`\"{}198012\char`\"{}.
\item Repetirá los pasos 5 a 12 para la lista de publicaciones obtenidas.
